О проекте
Urban Style - это современный интернет-магазин одежды с интуитивно понятным интерфейсом, фильтрацией товаров и интерактивными элементами. Проект создан как полнофункциональный веб-сайт для демонстрации возможностей frontend-разработки.

✨ Особенности
- Каталог товаров с категориями и фильтрацией
- Фильтрация по категориям и цене
- Корзина покупок с сохранением в localStorage
- Таблица размеров с интерактивным выделением
- Полностью адаптивный дизайн (мобильные устройства, планшеты, ПК)
- Быстрая загрузка и плавные анимации
- Форма обратной связи
